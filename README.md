# Phishing Free WiFi

## Introduction

A fake WiFi Hotspot, that imitates the facebook login page, asking for the user's credentials to do a check-in in order to get free WiFi access.

## Technologies

This project uses the following technologies:

-   [NodeJS](https://nodejs.org/en/)
-   [Typescript](https://www.typescriptlang.org/)
-   [JWT](https://jwt.io/)

### Client

-   [React](https://reactjs.org/)
-   [Vite](https://vitejs.dev/)

### Server

-   [Express](https://expressjs.com/)
-   [TypeORM](https://typeorm.io/#/)
-   [SQLite](https://www.sqlite.org/index.html)

### Raspberry Pi

-   [Raspberry Pi OS](https://www.raspberrypi.org/software/operating-systems/)
-   [Dnsmasq](http://www.thekelleys.org.uk/dnsmasq/doc.html)
-   [Hostapd](https://w1.fi/hostapd/)
-   Honorable mention to [TomHumphries](https://github.com/TomHumphries/RaspberryPiHotspot), with the guide on how to setup the Raspberry Pi as a WiFi Hotspot.
